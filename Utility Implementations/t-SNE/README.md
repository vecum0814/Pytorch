## t-SNE

t-SNE는 합성곱 신경망을 거쳐 나온 고차원의 Feature Map을 분석하기 위해 사용되는 차원 축소 기법 입니다. 


합성곱 신경망이 추출한 Feature Map들이 어떠한 분포를 가지고 있는지, 어떤 이미지들이 클래스의 경계 부분에 위치하는지 파악하는 것은 중요하기 때문에 일반적으로 마지막 합성곱 층에서 추출된 Feature Map을 가지고 분포를 그립니다.
